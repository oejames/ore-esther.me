3D Geo-Archive

content. Overview The New York Times 3D Article Geo-Archive is an interactive visualization project that maps articles from the New York Times Modern Love section onto a 3D globe. This project combines data fetching, geolocation, and 3D rendering to create an engaging way to explore Modern Love stories based on their geographical context. Tools & Technologies Three.js MongoDB Node.js Express D3.js HTML/CSS/JavaScript OpenStreetMap Nominatim API Key Skills 3D visualization API integration Geolocation processing Full-stack web development Database management Project Details Concept The project aims to create a visually appealing and interactive way to explore New York Times Modern Love articles based on their geographical settings. By mapping these stories onto a 3D globe, users can discover articles from different parts of the world, offering a unique perspective on love stories across cultures and locations. Implementation Data Fetching: Utilized the New York Times API to retrieve Modern Love articles, focusing on extracting relevant information such as titles, URLs, and location data. Geolocation Processing: Implemented a geolocation system using the OpenStreetMap Nominatim API to convert text-based locations into latitude and longitude coordinates. Database Integration: Designed and implemented a MongoDB database to store processed article data, including geocoded locations, for efficient retrieval and updates. 3D Globe Visualization: Developed a 3D globe using Three.js, rendering the Earth's surface and placing interactive points representing article locations. User Interaction: Created an interactive interface allowing users to hover over points to view article details and click to open the full article in a new tab. Server-Side Architecture: Built a Node.js/Express server to handle API requests, serve the frontend, and manage database operations.
